## Описание для ChatGPT App Store

**BeautyBot** — это интеллектуальный помощник для поиска и бронирования салонов красоты, парикмахерских и косметологических центров. Приложение использует AI для персонализированного поиска салонов на основе ваших предпочтений, местоположения и бюджета.

### Основные возможности

🔍 **Умный поиск салонов**
- Поиск по городу и типу услуги (стрижка, маникюр, педикюр, косметология и др.)
- Фильтрация по бюджету, рейтингу и доступности
- Сортировка по расстоянию от вашего местоположения
- Группировка результатов по сетям салонов

📋 **Детальная информация**
- Полная информация о каждом салоне: адрес, контакты, рабочие часы
- Рейтинг и отзывы клиентов
- Список услуг с ценами и описаниями
- Фотографии салона

📅 **Бронирование**
- Просмотр доступных временных слотов
- Бронирование с выбором услуги
- Автоматические уведомления салону

🎯 **Персонализация**
- Запоминание ваших предпочтений (любимый город, типы услуг)
- История поисков и бронирований
- Персонализированные рекомендации

### Как использовать

1. **Поиск салона**: Спросите ChatGPT найти салон красоты в вашем городе
   - "Найди салон красоты в Москве"
   - "Покажи парикмахерские рядом со мной"
   - "Ищу маникюр в Санкт-Петербурге до 2000 рублей"

2. **Получение информации**: Запросите детальную информацию о салоне
   - "Расскажи подробнее о салоне Оливер"
   - "Какие услуги есть в этом салоне?"
   - "Покажи отзывы о салоне"

3. **Бронирование**: Забронируйте время в салоне
   - "Забронируй стрижку на завтра в 15:00"
   - "Запиши меня на маникюр в салон Оливер"

4. **Проверка доступности**: Узнайте свободное время
   - "Когда свободен салон Оливер?"
   - "Покажи доступные слоты на этой неделе"

### Примеры использования

**Пример 1: Поиск салона**
```
Пользователь: Найди салон красоты в Москве для стрижки

ChatGPT: Нашел несколько салонов в Москве:
- Салон красоты Оливер (ул. Тверская, д. 10)
  Рейтинг: 4.7 ⭐ | Расстояние: 2.5 км
  Услуги: Стрижка женская (1500₽), Стрижка мужская (800₽)
  
Хотите узнать больше о каком-то салоне или забронировать время?
```

**Пример 2: Бронирование**
```
Пользователь: Забронируй стрижку в салоне Оливер на завтра в 15:00

ChatGPT: Отлично! Бронирование создано:
- Салон: Оливер
- Услуга: Стрижка женская
- Дата и время: 27 декабря, 15:00
- Телефон салона: +7-495-123-45-67

Салон получит уведомление и подтвердит бронирование.
```

### Технические детали

- **API Endpoints**: 7 основных endpoints для поиска, бронирования и получения информации
- **Персонализация**: Система запоминает ваши предпочтения для улучшения рекомендаций
- **Мониторинг**: Полное логирование всех запросов для обеспечения качества сервиса
- **Безопасность**: Все данные защищены, используется безопасная аутентификация

### Политика конфиденциальности

Мы собираем только необходимые данные для предоставления услуг:
- ID пользователя ChatGPT (для персонализации)
- История поисков и бронирований (для улучшения рекомендаций)
- IP-адрес и User-Agent (для безопасности и аналитики)

Все данные хранятся в зашифрованном виде и не передаются третьим лицам.

- Салон получит уведомление и свяжется с вами
