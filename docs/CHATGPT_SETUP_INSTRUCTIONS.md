# Инструкция по настройке BeautyBot для ChatGPT

## Для владельцев салонов красоты

### Шаг 1: Регистрация и создание бизнеса

1. Зарегистрируйтесь на платформе BeautyBot
2. Создайте профиль вашего салона:
   - Название салона
   - Адрес и контакты
   - Рабочие часы
   - Описание и специализация

### Шаг 2: Добавление услуг

1. Перейдите в раздел "Услуги"
2. Добавьте все услуги вашего салона:
   - Название услуги
   - Описание (SEO-оптимизированное)
   - Цена
   - Длительность
   - Категория

3. **Важно**: Добавьте `chatgpt_context` для каждой услуги:
   - Техники выполнения
   - Используемые материалы
   - Особенности услуги
   - Это поможет ChatGPT лучше описывать ваши услуги

### Шаг 3: Подключение к ChatGPT

1. Включите опцию "ChatGPT" в настройках бизнеса
2. Убедитесь, что ваш бизнес одобрен модерацией
3. Проверьте, что у вас активная подписка

### Шаг 4: Настройка контекста для ChatGPT

1. Добавьте `chatgpt_context` для вашего бизнеса:
   - Специализация салона
   - Особенности работы
   - Стиль и философия
   - Уникальные предложения

2. Пример хорошего контекста:
   ```
   Премиальный салон красоты с акцентом на натуральные материалы 
   и индивидуальный подход. Специализируемся на японских техниках 
   стрижки и окрашивания. Используем только профессиональную косметику 
   премиум-класса. Мастера с опытом работы от 5 лет.
   ```

### Шаг 5: Настройка уведомлений

1. Подключите Telegram бота для получения уведомлений о бронированиях
2. Или настройте WhatsApp для уведомлений
3. Проверьте, что уведомления приходят корректно

### Шаг 6: Проверка работы

1. Попросите ChatGPT найти ваш салон:
   ```
   "Найди салон [название вашего салона] в [ваш город]"
   ```

2. Проверьте, что информация отображается корректно:
   - Название и адрес
   - Контакты
   - Услуги и цены
   - Рейтинг и отзывы (если есть)

3. Протестируйте бронирование:
   ```
   "Забронируй [услуга] в [название салона] на [дата и время]"
   ```

## Для администраторов платформы

### Настройка API для ChatGPT

1. **Создание OpenAPI схемы**:
   - Файл: `docs/chatgpt_actions_openapi.yaml`
   - Уже создан и готов к использованию

2. **Настройка аутентификации**:
   - Создайте API ключи для ChatGPT
   - Настройте проверку ключей в endpoints
   - Добавьте ключи в OpenAPI схему

3. **Запуск миграций**:
   ```bash
   python src/migrate_add_chatgpt_context.py
   python src/migrate_add_chatgpt_sessions.py
   python src/migrate_add_chatgpt_requests.py
   ```

4. **Проверка endpoints**:
   - Убедитесь, что все endpoints работают корректно
   - Проверьте логирование запросов
   - Протестируйте статистику

### Мониторинг и аналитика

1. **Просмотр статистики**:
   - Endpoint: `GET /api/chatgpt/stats`
   - Требуется авторизация администратора
   - Параметры: `days`, `business_id`, `endpoint`

2. **Метрики для отслеживания**:
   - Количество запросов
   - Уникальные пользователи
   - Успешность запросов
   - Среднее время ответа
   - Топ бизнесов по запросам

### Публикация в ChatGPT App Store

1. **Подготовка материалов**:
   - Описание приложения (готово: `docs/CHATGPT_APP_DESCRIPTION.md`)
   - Политика конфиденциальности (готово: `docs/CHATGPT_PRIVACY_POLICY.md`)
   - Иконка приложения (512x512px)
   - Скриншоты использования

2. **Загрузка OpenAPI схемы**:
   - Импортируйте `docs/chatgpt_actions_openapi.yaml` в ChatGPT
   - Проверьте, что все функции работают корректно

3. **Тестирование**:
   - Протестируйте все сценарии использования
   - Проверьте обработку ошибок
   - Убедитесь в корректности ответов

4. **Подача заявки**:
   - Заполните форму в ChatGPT App Store
   - Приложите все необходимые материалы
   - Дождитесь модерации

## Рекомендации по оптимизации

### Для лучшей видимости в ChatGPT

1. **Оптимизация описаний**:
   - Используйте ключевые слова в описании салона
   - Добавьте информацию о специализации
   - Укажите уникальные особенности

2. **Контекст для ChatGPT**:
   - Заполните `chatgpt_context` для бизнеса
   - Добавьте контекст для каждой услуги
   - Это поможет ChatGPT лучше описывать ваш салон

3. **Отзывы и рейтинг**:
   - Регулярно отвечайте на отзывы
   - Поддерживайте высокий рейтинг
   - Это влияет на позицию в результатах поиска

4. **Актуальность информации**:
   - Регулярно обновляйте рабочие часы
   - Добавляйте новые услуги
   - Обновляйте цены
