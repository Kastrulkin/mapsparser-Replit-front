## Использование в коде

### CSS переменные
```css
background-color: hsl(var(--primary));
color: hsl(var(--primary-foreground));
```

### Tailwind классы
```tsx
<div className="bg-primary text-primary-foreground">
<div className="bg-success text-success-foreground">
<div className="border-border">
```

### Прямые значения
```css
/* Оранжевый primary */
background-color: #FF8C42;
background-color: hsl(18 100% 62%);
background-color: rgb(255, 140, 66);
```

## Цветовая схема

### Основная палитра
- **Primary**: Оранжевый (#FF8C42) - основной акцент
- **Neutral**: Серо-голубые оттенки для фонов и текста
- **Status**: Зеленый, желтый, синий, красный для статусов

### Контрастность
- Все цвета соответствуют WCAG AA стандартам контрастности
- Primary на белом: контраст 4.5:1
- Foreground на Background: контраст 7:1

---

## Градиенты (если используются)

Основной градиент можно создать на основе primary цвета:
```css
background: linear-gradient(135deg, #FF8C42 0%, #FF6B1A 100%);
```
