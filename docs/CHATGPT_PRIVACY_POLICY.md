# Политика конфиденциальности BeautyBot для ChatGPT

**Дата вступления в силу**: 26 декабря 2025  
**Последнее обновление**: 26 декабря 2025

## 1. Общие положения

BeautyBot ("мы", "наш", "сервис") уважает вашу конфиденциальность и обязуется защищать ваши личные данные. Эта политика конфиденциальности описывает, как мы собираем, используем и защищаем информацию при использовании нашего сервиса через ChatGPT.

## 2. Собираемая информация

### 2.1. Информация, предоставляемая ChatGPT
- **ID пользователя ChatGPT**: Уникальный идентификатор, предоставляемый ChatGPT для персонализации опыта
- **Параметры запросов**: Город, тип услуги, бюджет и другие параметры поиска
- **Данные бронирования**: Имя, телефон, email (если предоставлен), время бронирования

### 2.2. Автоматически собираемая информация
- **IP-адрес**: Для обеспечения безопасности и предотвращения злоупотреблений
- **User-Agent**: Информация о браузере/клиенте для технической поддержки
- **Время запросов**: Для анализа производительности и оптимизации сервиса
- **Метрики производительности**: Время ответа API, статус запросов

### 2.3. История взаимодействий
- **История поисков**: Город, услуга, количество результатов (для улучшения рекомендаций)
- **История бронирований**: ID салона, услуги, время (для персонализации)
- **Предпочтения**: Предпочтительный город и типы услуг (на основе истории)

## 3. Использование информации

Мы используем собранную информацию для:

### 3.1. Предоставление услуг
- Поиск салонов красоты по вашим критериям
- Создание и управление бронированиями
- Предоставление детальной информации о салонах

### 3.2. Персонализация
- Запоминание ваших предпочтений (город, типы услуг)
- Предложение релевантных результатов на основе истории
- Улучшение качества рекомендаций

### 3.3. Аналитика и улучшение сервиса
- Анализ использования API для оптимизации производительности
- Выявление популярных салонов и услуг
- Улучшение алгоритмов поиска

### 3.4. Безопасность
- Предотвращение злоупотреблений и мошенничества
- Защита от DDoS-атак и других угроз
- Обеспечение безопасности данных

## 4. Передача данных третьим лицам

### 4.1. Салоны красоты
При создании бронирования мы передаем салону следующую информацию:
- Имя клиента
- Телефон клиента
- Email клиента (если предоставлен)
- Время и дата бронирования
- Название услуги (если указана)
- Дополнительные заметки (если указаны)

**Цель**: Обеспечение связи между клиентом и салоном для подтверждения бронирования.

### 4.2. Сервисы уведомлений
Мы используем следующие сервисы для отправки уведомлений салонам:
- **Telegram**: Для отправки уведомлений через Telegram ботов
- **WhatsApp**: Для отправки уведомлений через WhatsApp (если настроено)

**Цель**: Информирование салонов о новых бронированиях и запросах поддержки.

### 4.3. Мы НЕ передаем:
- ID пользователя ChatGPT третьим лицам
- Историю поисков и бронирований
- IP-адреса и другую техническую информацию
- Данные для рекламных целей

## 5. Хранение данных

### 5.1. Период хранения
- **История поисков**: Хранится до 50 последних запросов
- **История бронирований**: Хранится до 20 последних бронирований
- **Логи запросов**: Хранятся для аналитики и безопасности
- **Сессии пользователей**: Хранятся для персонализации

### 5.2. Безопасность хранения
- Все данные хранятся в зашифрованной базе данных
- Используется безопасное подключение (HTTPS) для всех API запросов
- Регулярные резервные копии данных
- Ограниченный доступ к данным только для авторизованного персонала

## 6. Ваши права

Вы имеете право:

### 6.1. Доступ к данным
- Запросить информацию о том, какие данные мы храним о вас
- Получить копию ваших данных в структурированном формате

### 6.2. Удаление данных
- Запросить удаление вашей истории поисков и бронирований
- Удалить ваши предпочтения и персонализацию

### 6.3. Отказ от персонализации
- Отказаться от сохранения истории взаимодействий
- Отказаться от использования предпочтений для рекомендаций

**Для реализации ваших прав**: Обратитесь к нам через функцию "Запросить поддержку" в ChatGPT или напишите на support@beautybot.ai

## 7. Cookies и отслеживание

Мы не используем cookies или другие технологии отслеживания для сбора информации о вас. Вся информация собирается только через API запросы от ChatGPT.

## 8. Безопасность

Мы принимаем следующие меры для защиты ваших данных:

- **Шифрование**: Все данные передаются по защищенному соединению (HTTPS)
- **Аутентификация**: Используется безопасная система аутентификации для доступа к API
- **Мониторинг**: Постоянный мониторинг системы на предмет подозрительной активности
- **Обновления безопасности**: Регулярное обновление системы безопасности

## 9. Изменения в политике конфиденциальности

Мы можем периодически обновлять эту политику конфиденциальности. О существенных изменениях мы уведомим вас через:
- Обновление даты "Последнее обновление" в начале документа
- Уведомление в ChatGPT (если применимо)

Рекомендуем периодически просматривать эту политику для ознакомления с актуальной информацией.

Если у вас есть вопросы или замечания по поводу этой политики конфиденциальности:

## 11. Соответствие законодательству

Мы соблюдаем требования:
- **GDPR** (General Data Protection Regulation) для пользователей из ЕС
- **Федеральный закон № 152-ФЗ** "О персональных данных" для пользователей из России
- Другие применимые законы о защите данных

**Примечание**: Эта политика конфиденциальности применяется только к использованию BeautyBot через ChatGPT. Использование веб-сайта или мобильного приложения может регулироваться отдельной политикой конфиденциальности.
